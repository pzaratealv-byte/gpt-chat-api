
# Chatbot PHP con OpenAI para Marketing Digital

Proyecto de chatbot web desarrollado en **PHP**, conectado a la **API de OpenAI**, pensado para responder consultas sobre marketing digital desde una interfaz sencilla y desplegable en un servidor **Plesk**.

El sistema funciona sin base de datos y puede instalarse fácilmente subiendo los archivos al hosting mediante **FTP**.

---

## Descripción

Este proyecto permite integrar un chatbot en una página web para que el usuario pueda hacer preguntas relacionadas con marketing digital y recibir respuestas generadas por inteligencia artificial.

Está diseñado para ser una solución simple, funcional y fácil de desplegar en entornos compartidos o servidores con **Plesk**, sin depender de frameworks pesados ni configuraciones complejas.

---

## Características

- Interfaz web sencilla y directa
- Backend en PHP
- Integración con la API de OpenAI
- Sin base de datos
- Despliegue rápido en hosting con Plesk
- Estructura ligera y fácil de mantener
- Proyecto adaptable a otros sectores además de marketing digital

---

## Tecnologías utilizadas

- **PHP**
- **HTML5**
- **CSS3**
- **JavaScript**
- **OpenAI API**
- **Plesk** para despliegue
- **FTP** para subida de archivos
